# Pipe Hunter Game

A fun Flappy Bird-style game with powerups, portals, weather effects, and sarcastic messages.

## Controls

- Arrow keys: Move
- Shift: Dash
- Collect coins for powerups
- Avoid enemy

## Features

- Levels and scoring
- Confusion mode and magnet powerups
- Teleport portals
- Cloud background
- Rage messages on game over
